# Book Tracker App

### Created by:

Mary Sanni

## Description:
This app serves as a book borrowing tracker, allowing me to keep track of the books I borrowed from the library.

### Technologies Used

- HTML
- CSS
- JAVASCRIPT
- NODE.JS
- REACT
- PYTHON
- MONGODB
- MATERIAL UL
- FRAMER MOTION
